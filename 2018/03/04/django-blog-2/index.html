配置数据库

django支持多种数据库如：oracle,mysql,sqlite等。
这里使用mysql数据库。
配置参数
‘ENGINE’：’django.db.backends.mysql’
‘NAME’：数据库名称
‘USER’：数据库用户
‘PASSWORD’：数据库密码
‘HOST’：数据库主机
‘PORT’：数据库监听端口
如图:

ubuntu下安装mysql

#安装mysql
sudo apt-get install mysql-server
sudo apt isntall mysql-client
sudo apt install libmysqlclient-dev
#创建数据库并授权
mysql -uroot -p
>mysql create database blogtt;
>mysql grant all privileges on blogtt.* to 'blogtt'@'%' identified by 'blogtt';
>mysql flush privileges;
